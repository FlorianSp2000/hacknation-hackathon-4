# Changelog

## v0.1.0

- Multi-model video inference pipeline (detection, segmentation, VLM)
- Registry-based model system with `@register` decorator
- Models: YOLOv8, YOLO11, SAM3, LFM2.5-VL-1.6B
- Pipeline with frame skip, VLM skip, sequential offload
- Streamlit UI: upload, inference, visualization, export
- Singularity container + SLURM deployment
